Web Projects

Wādoru

This was my first React project, created in 2023 while I was learning the framework. It's an anime guessing game inspired by the NBA Poeltl game. The game fetches an anime title from the MyAnimeList (MAL) free API, which users try to guess. I used Framer Motion to add subtle visual animations, and Firebase as the database to ensure that users around the world are guessing the same anime title while avoiding repetition.

https://anime-wadoru.web.app

Public Market Stall Renting System

I developed this MERN stack project last year as part of a client's thesis project, tailored specifically for our local public market. The system streamlines the management of stalls, vendors, and rental payments, allowing administrators to efficiently handle vendor profiles, assign stalls, and track rentals. For on-site payment collection, I created a simple React Native app that enables collectors to record transactions and print receipts instantly using a mobile printer.

Cafe Leone Restaurant Employee Management System

This laravel application I created last year is designed to streamline the management of employee schedules and leave requests in a restaurant setting. It allows employees to request schedule exchanges with their colleagues and submit leave requests directly through the platform. Admins have access to a dedicated portal where they can efficiently review, approve, or reject both schedule exchanges and leave submissions, ensuring smooth operations and improved communication within the workplace.

https://github.com/JayMarPabayo/hris-app.git

Financial Assistance Document Management System

I also developed this laravel project last year as part of the client's thesis project. It is designed to streamline the application process for various types of assistance, including Burial Assistance, Medical Billing Assistance, and more. The platform allows users to submit required documents online and receive updates via email. Authorized staff members can securely review and verify applications, ensuring that all submitted documents meet the necessary requirements.
